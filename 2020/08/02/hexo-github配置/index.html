一、背景

  • gitpage

gitpage是github提供的一种利用github仓库快速搭建的博客 通过dspure@github.io的方式即可访问

  • hexo

hexo是网页部署的神器，利用hexo可以快速将自己设计的网页部署到github上，而且对比gitbook使用服务器部署，hexo方便多了，而且对我这种不会前端的人来说，hexo的主题是真的不错

  • 简短的絮叨

我这种前端小白，配置大半天才配好，挺不容易的，现在想想挺简单，github建个仓库配置一下，hexo配置一下就好了，哈哈哈

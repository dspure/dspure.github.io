mysql学习笔记

2020 7 12

mysql -u root -p liu123 //登录

  • select version(); //查看版本
  • show databases; //查看库
  • creat database xx; / drop database xx; //创建删除库
  • use xx; //选择库
  • show tables; //查看表
  • 数据类型

https://www.runoob.com/mysql/mysql-data-types.html

  • 创建/删除表

  • 添加数据

  • 显示全部

  • select * from mytable;

  • where

    • where

    • select * form mytable where id < 10 ;

    • select age form mytable where score > 60;

  • update

  • delete

  • LIKE

    • select * from mytable where name like "robot%";

  • union

    • select name from mytable union select name from mytable2;
    • select name from mytable union all select name from mytable2; //统计相同字段个数
    • union all显示重复的 union不显示

  • order by asc/desc

    • select * from mytable order by score;

  • group by

    • select name from mytable group by name; //查询字段要有group by内容

  • [inner] join … on … //inner可省略 组合多个表内容

    • select name, age, father, mony, familycount from mytable inner join mytable2 on mytable.id = mytable2.m_id;
    • 等价于
      • select name, age, father, mony,familycount from mytable,mytable2 where mytable.id = mytable2.m_id;

  • is null / is not null //判断字段是否为空

    • ifnull(列名，值) //替换值
      • select ifnull(name,’xxx’) from mytable;

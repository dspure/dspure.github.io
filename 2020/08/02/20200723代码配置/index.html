NC 、 LDAP 、 邮箱 激活

一、工作环境配置

  • 常用工具安装

  • virtualbox 、chrome

  • markdown编辑器

    • https://typora.io/#linux

  • shutcut tool

    • shutter

    • 快捷键设置：https://jingyan.baidu.com/article/4b52d702db5482fc5c774bf0.html

  1. git设置名字和邮箱

  2. 配置java8

  1. 下载repo，添加到PATH

  • git clone https://aosp.tuna.tsinghua.edu.cn/git-repo/

  • 修改 /etc/profile 或者 /home/ts/.bashrc 添加 export PATH=”$PATH:repo执行文件所在目录”

  • 修改~/git-repo/repo将REPO_URL替换为清华镜像地址，以避免下载android源码时可能出现的无法连接到 gerrit.googlesource.com问题。

  1. ssh-keygen -t rsa -C “email” 生成密钥

  2. gerrit 添加公钥

二、代码同步

一般公司会给代码同步地址

repo init -u 代码地址

1. AP编译
2. BP编译
3. 整编（AP + BP 但是要将AP代码放到BP 目录名字要跟脚本相同
4. 编译单个模块
